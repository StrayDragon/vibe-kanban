## Overview

AI coding agents are increasingly writing the world's code and human engineers
now spend the majority of their time planning, reviewing, and orchestrating
tasks. Vibe Kanban streamlines this process, enabling you to:

- Switch between different coding agents
- Orchestrate multiple coding agents in parallel or in sequence
- Review work quickly and start dev servers
- Track the status of tasks your agents are working on
- Centralize MCP configuration for coding agents
- Open projects remotely via SSH when running on a remote server
